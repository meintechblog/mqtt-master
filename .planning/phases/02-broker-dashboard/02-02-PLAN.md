<task type="auto">
  <name>Task 1: WebSocket client module and stat card component</name>
  <files>webapp/js/lib/ws-client.js, webapp/js/components/stat-card.js</files>
  <action>
    **webapp/js/lib/ws-client.js:**
    Create a WebSocket client module that connects to the backend and exposes Preact signals for reactive UI updates.

    - Import `signal` from `@preact/signals`
    - Export signals: `dashboardState = signal({ data: {}, topics: {} })`, `brokerConnected = signal(false)`, `wsConnected = signal(false)`
    - Export `connectDashboardWs()` function:
      - Determine WS URL from `window.location`: `ws://${location.host}/ws/dashboard` (or `wss://` if `location.protocol === 'https:'`)
      - Create `new WebSocket(url)`
      - On open: set `wsConnected.value = true`
      - On message: parse JSON, switch on `msg.type`:
        - `"sys_state"`: update `dashboardState.value = { data: msg.data, topics: msg.topics }`
        - `"connection_status"`: update `brokerConnected.value = msg.connected`
      - On close: set `wsConnected.value = false`, schedule reconnect
      - On error: close socket (triggers onclose -> reconnect)
      - Reconnect logic: exponential backoff starting at 1s, doubling to max 30s, reset to 1s on successful open
    - Export `disconnectDashboardWs()` to close cleanly and cancel reconnect timer
    - Auto-connect on module import (call `connectDashboardWs()` at module level)

    **webapp/js/components/stat-card.js:**
    Create a reusable stat card component for dashboard metrics.

    - Import `html` from `htm/preact`
    - Export `StatCard({ label, value, unit })` component
    - Renders a `.ve-card` div with:
      - `.stat-value` div: large text (the value), styled at 28px font-size, white color
      - `.stat-unit` span: inline with value if unit provided (e.g., "KB"), smaller dim text
      - `.stat-label` div: below value, 13px, `--ve-text-dim` color
    - If value is null/undefined, display "--" as placeholder
  </action>
