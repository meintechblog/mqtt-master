       - Export class `ConfigService` with constructor(configPath)
       - DEFAULTS object matching config/default.json values exactly
       - async load(): check existsSync(this.configPath), if missing return silently. If exists, readFile + JSON.parse, wrap in try/catch, on error console.warn and keep defaults. On success, deep merge file over defaults.

    5. Create `server/services/mqtt-service.js`:
       - Export class `MqttService` extends EventEmitter (from 'node:events')
       - constructor(brokerUrl): set this.brokerUrl, this.client = null, this.connected = false
       - async connect(): return Promise. Call mqtt.connect(this.brokerUrl, { clientId: `mqtt-master-${Date.now()}`, clean: true, reconnectPeriod: 5000 }). On 'connect' event: set this.connected = true, emit 'connected', resolve(). On 'error': emit 'error'. On 'close': set this.connected = false, emit 'disconnected'. On 'message': emit 'message' with {topic, payload: payload.toString(), timestamp: Date.now()}. setTimeout 5000ms: if not connected, resolve() anyway (don't block startup).
       - subscribe(topic): if this.client, call this.client.subscribe(topic)
       - publish(topic, payload, opts): if this.client, call this.client.publish(topic, payload, opts)
       - isConnected(): return this.connected

    6. Create `server/index.js`:
       - Import Fastify (default), fastifyStatic (default from @fastify/static), fastifyWebSocket (default from @fastify/websocket)
       - Import fileURLToPath from 'node:url', dirname/join from 'node:path'
       - Import ConfigService from './services/config-service.js'
       - Import MqttService from './services/mqtt-service.js'
       - Compute __dirname from import.meta.url
       - Export async function start(opts = {}):
         - configPath = opts.configPath || '/opt/mqtt-master/config.json'
         - const config = new ConfigService(configPath)
         - await config.load()
         - const app = Fastify({ logger: { level: config.get('logLevel', 'info') } })
         - Register @fastify/static with root: join(__dirname, '..', 'webapp'), prefix: '/'
         - Register @fastify/websocket
         - const mqttService = new MqttService(config.get('mqtt.broker', 'mqtt://localhost:1883'))
         - await mqttService.connect()
         - app.decorate('mqttService', mqttService)
         - app.decorate('configService', config)
         - app.setNotFoundHandler((req, reply) => reply.sendFile('index.html'))
         - const port = opts.port || config.get('web.port', 3000)
         - const host = opts.host || '0.0.0.0'
         - await app.listen({ port, host })
         - return app
       - If not in test mode (no opts.test): call start() at bottom, catch err => console.error + process.exit(1)
         Use: `if (process.argv[1] && fileURLToPath(import.meta.url) === process.argv[1]) { start().catch(...) }`
